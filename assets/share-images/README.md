# 分享图片目录

此目录存放分享功能所需的图片资源。

## 默认分享图片

当没有动态生成分享图片时，会使用 `default-share.png` 作为默认分享图。

### 要求

- 文件名: `default-share.png`
- 推荐尺寸: 500 x 400 px (5:4 比例)
- 格式: PNG (支持透明背景)
- 大小: < 200KB

### 建议

建议使用设计工具创建一张包含以下元素的分享图：
- GachaPlaner 品牌 Logo
- 简洁的背景色（建议使用莫兰迪暖色系 #F5F3F0）
- 简短的产品介绍文字

### 注意

如果不提供此文件，分享功能会尝试动态生成分享图片。
