# Story 5.5: 游戏数据版本标注

Status: ready-for-dev

## Story

As a 用户,
I want 看到游戏数据版本信息,
So that 了解数据时效性.

## Acceptance Criteria

**Given** 游戏配置包含版本信息
**When** 用户查看游戏详情或设置页
**Then** 显示游戏数据版本号（如"原神数据 v1.0.0"）
**And** 显示数据更新日期（如"更新于2026-04-01"）
**And** 版本信息醒目但不干扰主要内容
**And** 支持查看所有游戏版本列表
