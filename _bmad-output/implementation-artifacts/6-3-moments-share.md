# Story 6.3: 朋友圈分享功能

Status: review

## Story

As a 用户,
I want 分享规划结果到朋友圈,
So that 展示我的抽卡规划给更多人.

## Acceptance Criteria

**Given** 规划结果图片已生成
**When** 用户点击分享朋友圈按钮
**Then** 图片保存到本地相册
**And** 提示用户去朋友圈发布
**And** 分享图片包含完整规划信息
**And** 图片尺寸适配朋友圈展示（正方形或宽图）
**And** 显示分享引导提示

## Dev Notes

朋友圈分享功能，由于微信小程序API限制，需要通过保存图片到相册的方式实现。

### 实现方案

微信小程序不支持直接调用分享朋友圈API，实现流程：

1. 生成适配朋友圈的图片
2. 将图片保存到用户相册
3. 显示分享引导弹窗，提示用户去朋友圈发布
4. 提供发布步骤说明

### 朋友圈图片尺寸建议

- 正方形：750 x 750 px
- 宽图：1080 x 1920 px（适合纵向展示）
